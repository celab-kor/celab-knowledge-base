## 사용 방법

### Claude에게 용어 수정 요청하기
```
TERMS.md의 용어 기준으로 이 텍스트를 교정해줘: [텍스트]
```

### 새 용어 추가하기
이 파일을 직접 편집한 후 저장하면 됩니다.  
파일 위치: `celab-knowledge-base/TERMS.md`

### 자동 반영 조건
Claude Code 세션에서 `celab-knowledge-base` 폴더를 작업 디렉토리로 사용하거나,  
CLAUDE.md에 이 파일을 참조하도록 설정하면 자동으로 읽힙니다.
