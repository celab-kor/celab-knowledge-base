권장 전략: HTML 대시보드 + Obsidian 노트 병행

인터랙티브 HTML은 7,415건의 Q&A 전체를 로컬에서 검색·필터·시각화하는 메인 플랫폼으로, Obsidian Vault는 개념 간 관계를 마크다운으로 정리하고 그래프 뷰로 내부화하는 지식 내면화 레이어로 운영합니다. 두 도구 모두 100% 로컬 실행이 가능하여 데이터 유출 위험이 없습니다.

인터랙티브 HTML 담당 역할

  • 7,415건 전체 데이터 검색 및 필터링
  • 온톨로지 노드 관계 D3 Force Graph 시각화
  • 카테고리·키워드·난이도 다중 필터 UI
  • 브라우저만 있으면 실행 — 설치 불필요
  • JSON 파일로 데이터 업데이트 가능

Obsidian Vault 담당 역할

  • 핵심 개념 노트 (MOC: Map of Content)
  • [[Wikilink]] 로 개념 간 관계 명시
  • Graph View 로 지식 클러스터 파악
  • Dataview 플러그인으로 태그 기반 검색
  • 전문가 학습 일지 및 리뷰 노트 관리
