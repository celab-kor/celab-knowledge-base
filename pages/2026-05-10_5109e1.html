안건 1 — 인화공 DB 분석 및 온톨로지 활용 방안

📊 인화공 데이터 분석 현황
지표수치비고
인화공 전체 스레드7,415건인터넷 화학공학회 DB
스테파노님 직접 답변6,474건20~30년 누적 실무 응답
온톨로지 후보838건AutoFlow 구축 의미 있는 스레드
FAQ / Calculation 비율67.6%단순 계산 기준 답변
1차 파일럿 선별 목표100건 → 500건전문가 리뷰 후 점진 확대
💡 데이터 가치 판단
"Rule & Regulation + 안목지" 기준의 고퀄리티 응답이 온톨로지 전체 그릇의 80~90%를 채울 수 있을 것으로 추정. 단, 맹목적 AI 학습 주입이 아닌 도메인 카테고리 분류 → 전문가 1차 검수 → JSON 스키마 정제 절차 필요.

분류 카테고리(예시): 이퀴프먼트·프로세스·파이핑·세이프티·Calculation / 설계 기준 / 트러블슈팅 / 운전 경험 / 레슨런드

💡 안건 1 인사이트
  • ◆
    단순 학습이 아닌 온톨로지 구조화가 핵심. 7천여 건의 QA를 "AI에 때려넣는" 방식이 아닌, 노드·엣지·관계로 구조화해야만 99% 정확도 달성 가능. 스키마 설계 단계부터 도메인 전문가 참여 필수.
  • ◆
    Calculation 67%는 LLM 환각 방지의 핵심 자산. 업계 Rule & Regulation을 명확히 학습시키면 계산 기반 질의에서 경쟁사 대비 차별화 가능.
  • ◆
    838건이 '완성'이 아니라 '큰 그릇'을 그리는 베이스라인. 해외 커뮤니티 DB 등 추가 소스와 병합 시 온톨로지 커버리지 확장 여지 충분.
  • ◆
    데이터 활용 법적 리스크 선제 검토 필요. 질의자 개인정보, 스테파노님 동의 서류화, 회사 종료 시 삭제 조항 등 법무 의견서 확보 선행.
  • ◆
    마일스톤 기반 정확도 로드맵 수립이 투자자·고객 신뢰 형성 핵심. 26년 말 90%, 27년 99% 목표 설정 시 전체 온톨로지 "그릇 크기"를 먼저 정의해야 계획 유효.

안건 2 — 투트랙 전략: PAWG(내부) vs. 인화공 생태계(외부)

🔒 트랙 1 — PAWG (내부)
Plant AI Working Group · 비공개 자문단
  • 온톨로지 구조 설계 · 검증
  • POC 프로젝트 실시간 자문
  • AI 개발자 ↔ 도메인 전문가 중간 통역
  • NDA 체결, 내부 기밀 공유 가능
  • 정기 회의로 마일스톤 관리
이종현 부장님 리딩 스테파노님 장현기 선생님
🌐 트랙 2 — 인화공 생태계 (외부)
Domain Product Council · 공식 생태계
  • 플랜트 AI 지식 네트워킹
  • 시니어 엔지니어 포모(FOMO) 해소 콘텐츠
  • 아티클·매거진 발행 (이유룡 교수님 필두)
  • 스냅스케일 브랜딩 · 신뢰 구축
  • 인화공 DB·회원 데이터 연계 협의
인화공 독립 운영 스냅스케일 기술 파트너
🔄 트랙 간 연계 및 논의 쟁점
⚠️ 대표님 우려 사항
현재 KPI는 "POC 정확도 향상 + 대기업 수주 확대"에 집중 필요. 트랙 2(교육 플랫폼, 커뮤니티 운영 등)가 지금 당장 이를 가속하는지 불분명. 리소스 분산 리스크.
📌 진행자 입장
트랙 2는 현재 직접 투입 X. 그러나 인화공 생태계에서 생산되는 QA·데이터·네트워크가 트랙 1 온톨로지 완성도를 높이는 간접 파이프라인. 셀럽화공연구소(주니어 교육 파이프라인)는 당장 불필요 → 현재 제외, 추후 재검토.
✅ 합의된 방향
트랙 1(PAWG) 구성은 즉시 진행. 트랙 2(인화공 공식 파트너십)는 스테파노님·장현기 선생님 미팅 결과를 보고 인화공 DB 사용 승인 여부 우선 확인 후 범위 결정.
💡 안건 2 인사이트
  • ◆
    자문단 영입 인센티브 설계가 선순환의 열쇠. 단순 자문료보다 레퓨테이션(아티클·크레딧), 매출 수수료 연동, 스톡옵션이 기여 촉진 구조를 형성함. 정량적 모델링 필요.
  • ◆
    외부 글로벌 인플루언서는 "지금은 아니고, 오픈 시점"에 활용. 제품 오픈 전 광고성 협업은 비용 대비 효과 낮음. 레퍼런스용 1회 소개 수준이 적정.
  • ◆
    배관·기계·전장 분야 전문가 추가 영입 시급. 현 자문단이 공정 중심 → 견적/멀티 디서플린 수주를 위해 세이프티 로직, 전장 분야 보완 필요.
  • ◆
    아스펜 유니버시티 모델(자격증·생태계 선점)이 중장기 레퍼런스. 현재는 아니지만, 교육 콘텐츠 누적 → 인증 체계 → 글로벌 생태계 표준화 방향성 유효.

🤝 내일 미팅 — 스테파노님 협력 제안 3가지 시나리오
시나리오 A — 온톨로지 라이선스 협약

스테파노님 QA DB를 플랜트 온톨로지 핵심 데이터로 활용. 개인정보 제거 후 공식 지식 기여자로 명기. 누적 기여에 대한 리테이너 보상 + 레퓨테이션 크레딧 제공.

시나리오 B — PAWG 비공식 자문 그룹 합류

분기 1회 전체 회의 참석 + 월 1~2회 온라인 세션. 제품 반영 결과 월간 보고. Domain Product Council 공식 타이틀 부여. 리테이너(컨설팅 자문료) 지급.

시나리오 C — 인화공 독립 파트너십

인화공 독립성 완전 보장 + 스냅스케일이 기술 파트너로 플랫폼 고도화 지원(AI 검색, 커뮤니티 도구). 회원 DB 마케팅 활용 협의. 공식 기술 협력 파트너 인증.

핵심 메시지: "20년 동안 혼자 싸우신 지식이 AI 시대 플랜트의 심장이 됩니다. 합류가 아닌, 20년 역사가 다음 세대를 가르치는 기반이 되는 것입니다."
🔧 내일 미팅 — 장현기 선생님 협력 제안 전략
접근 방식: 설득보다 역할 부여
"저희는 이렇게 하겠습니다. 도움 주세요"보다 "선생님이라면 이 구조를 어떻게 설계하시겠습니까?" → 고민하게 만드는 방식
  • 🎯
    인화공 데이터 분석 현황 공유 → 온톨로지 도메인 분포(공정/배관/장치/제어) 피드백 요청
  • 🎯
    PAWG 온톨로지 아키텍처 자문단 공식 참여 요청 (월 1~2회 온라인 리뷰)
  • 🎯
    인센티브: 스톡옵션 + 개발 모듈 매출 수수료 연동 제안 (기여도 % → 수수료 % 비례 구조)
  • 🎯
    장기 비전: 사우디 근무 이후 한국에서의 플랜트 AI 커리어 연계 가능성 제시
  • 🎯
    파일럿 100건 온톨로지 리뷰 참여 타진 → 12주 로드맵으로 연결
💡 안건 3 인사이트
  • ◆
    각 자문단의 동기부여 구조는 완전히 다르다. 스테파노님 → 레퓨테이션·역사적 기여. 장현기 선생님 → 매출 수수료·기술 크레딧. 이유룡 교수님 → IP·논문·계약. 획일적 인센티브 설계는 실패.
  • ◆
    워스트 케이스(아무도 안 옴) 전제 하에 플랜 B 필요. 관계 기반으로 쌓아온 신뢰가 있으나, 도메인 전문가가 스타트업을 위해 자발적으로 깊이 기여할 유인은 제한적. 명확한 인센티브가 없으면 형식적 참여에 그칠 위험.
  • ◆
    아스펜 테크 레거시 전략(학술 논문 → 업계 표준화)이 이유룡 교수님 협력의 핵심 이유. 연구 논문과 AI 제품이 연동되면 글로벌 생태계 표준 선점 가능.
  • ◆
    스테파노님 상업화 경계가 최대 변수. 인화공 DB가 상업적으로 활용된다는 인식이 생기면 협력 전체가 무너질 수 있음. 독립성 보장 조항을 계약서에 선명히 명시 필요.

안건 4 — 이유룡 교수님 계약 구조 업데이트

항목교수님 요청협의 결과
스톡옵션 희석다이류션(희석) 없이 유지 요청희석 없이 유지 방향 동의
자문료 수준20%(초안) → 50% 요청박교부장님 연봉 50% 수준 협의
자문료 연동시리즈 A 연봉 인상 시 자동 연동절대 최대금액(방어선) 별도 설정 요구
지급 주기월 지급 → 연 2~3회연 2~3회 지급 (보험 비적용)
특허 공동 출원스냅스케일과 공동 출원 제안내부 검토 필요 (85:15 분할 제시)
계약 초안—회사 법인 쪽 초안 요청, 금요일 미팅
⚠️ 추가 검토 사항
특허 공동 출원 및 특허 과제 협력(스냅스케일 + 교수님 연구실) → 대표 최종 의사결정 필요. 계약 초안에 종료 시 데이터 삭제 조항, 교수님 겸업 금지 충돌 여부 법무 검토 병행.
✓

핵심 결정사항

결정 01

투트랙 전략 기본 방향 유지. PAWG(내부 자문) + 인화공 생태계(공식 파트너십)의 이중 구조로 진행. 셀럽화공연구소 주니어 교육 파이프라인은 현 단계 제외.

결정 02

스테파노님·장현기 선생님 미팅(익일)에서 인화공 DB 사용 동의 선확보 후 본격 활용 결정. 온라인 세션 빈도·자문료 등 세부 조건은 미팅 결과 기반으로 재협의.

결정 03

이유룡 교수님 계약: 스톡옵션 희석 없음 + 자문료 50% + 연 2~3회 지급 방향 합의. 절대 최대금액 방어선 조항 삽입. 특허 공동 출원은 별도 검토.

결정 04
