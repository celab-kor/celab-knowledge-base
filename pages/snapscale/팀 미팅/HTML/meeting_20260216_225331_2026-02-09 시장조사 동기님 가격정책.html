시장조사 방법론 및 자료 소스 현황 공유
정보 공유

🔍 핵심 논의 내용

  • 동현이 수행한 시장조사는 총 3개 PPT 파일과 ZIP 압축파일(220MB)로 구성되어 있으며 카톡으로 재공유됨
  • 시장조사 1: 석유화학 도메인 기초 개념(업/다운스트림, 사이클 지표) + 정유사·한국 석유화학사 대응 전략 + PwC·EY 자료 기반 2030 전망 → 넓고 얕게 조사
  • 시장조사 2: 한국플랜트산업협회 자료 기반 EPC업체별 비교 및 중동·남미 플랜트 대응 분석 → 대회 준비로 인해 미완성 상태
  • 시장조사 3 (탐샘솜용): 국내 EPC 상위 15개사 매출 분석, DART 연간 사업보고서 활용, 석유화학·정유 섹터 집중

📚 주요 자료 출처

🔗 구글링 + 증권사 리포트 🔗 PwC / EY 컨설팅 자료 (GPT 번역·요약) 🔗 한국플랜트산업협회 자료실 🔗 DART 연간 사업보고서 🔗 OPEC World Oil Outlook (328p) 🔗 증권사 리서치센터 (하나·LS 등)

💡 안건 Insight

📐
도메인 비전문가의 구조적 한계 인식

조사팀 전원이 석유화학 도메인 지식 부재로 1차 조사는 기초 개념 파악 수준에 머물렀음. 이는 추후 심층 조사 시 도메인 전문가 협력의 필요성을 시사함.

✅
공신력 있는 복합 소스 활용 전략

구글링→증권사 리포트→컨설팅 펌→협회 자료로 이어지는 계층적 소스 검증 방식을 사용하여 자료 신뢰도를 확보함. AI(GPT)를 활용한 영문 자료 번역·요약도 효율적으로 운영됨.

⚠️
한국플랜트산업협회 자료실 접근 제한

현재 유료 회원 로그인 필요로 자료실이 막혀있음. 다만 동현이 사전에 다운받아 ZIP 파일로 보관 중이므로 즉시 활용 가능. 향후 공식 가입 여부 검토 필요.

🚨
드라이브 데이터 손실로 남미·중동 자료 부재

노트북 교체 과정에서 약 3GB 분량의 남미·중동 관련 핵심 자료가 유실됨. 카톡 대화 이력에서 복구 가능성 있으나 확인 필요.

2
국내 EPC 상위 15개사 재무 분석 방법론 검토
심층 분석

🔍 핵심 논의 내용

  • DART 금융감독원 공시 시스템에서 연간 사업보고서(2024~2025년도) 기반으로 15개사 매출 분석
  • 분석 항목: 화공 부문 매출액, 수주잔액(수주금액·공정률), 부문별 이익현황
  • 삼성 E&A만 화공 매출 별도 산정 가능. 타사는 원전·발전과 통합 계상하는 경향
  • 회계 처리 방식 차이(균등배분 vs 완공률 기준)로 연간 매출 비교의 엄밀성 한계 존재
  • 에코플랜트는 해당 기간 화공 수주 실적 전무 확인
  • 특허 조사 시도(현대엔지니어링, 삼성) → 도메인 한계로 분석 미완성

🧮 마진율 산출 불가 원인 분석

  • 대손충당금 규모의 불확실성 및 회사별 상이한 회계 처리
  • 환율 헤지 (환 파생상품) 손익이 실질 마진에 미치는 영향 불투명
  • EPC 특성상 수주-완공 기간이 수년에 걸쳐 있어 연간 인식 매출의 의미 해석 어려움
  • 재무전략기획 담당자 인터뷰 3건 시도 → 전원 거절 (영업기밀 이슈)

💡 안건 Insight

🏗️
DART 기반 분석은 탐색적 규모 파악 수준으로 한계 명확

공개 재무데이터만으로는 정밀한 세그먼트 마진율 산출이 불가능함. 대손충당금, 환헤지 손익 등 비영업 항목 영향이 크기 때문으로, 보완적 정보 소스(전문가 인터뷰, 내부 IR 자료 등) 필수.

💎
삼성 E&A가 벤치마크 기준점으로 활용 가능

15개사 중 화공 부문 별도 산정이 가능한 유일한 기업. BM·프라이싱 기초 데이터 수집 시 삼성 E&A 중심으로 심층 분석 우선 진행 권고.

🔑
증권사 리서치 리포트가 예측 데이터의 최적 소스

기업 실사 기반의 애널리스트 리포트(하나·LS 등)는 실적 예측 및 수주 전망 관련 가장 신뢰도 높은 공개 자료. 향후 BM 수립 시 주요 소스로 적극 활용 필요.

3
BM 모델 및 가격정책 수립 방향성 논의
전략 논의

🔍 핵심 논의 내용

  • 현재 개발 중인 제품은 "기존 소프트웨어 공급"이 아닌 "엔지니어링 접근 방식 자체를 변경"하는 게임 체인저 포지셔닝
  • 가격 산정 방향: 프로젝트 수주 건당 비용 산정 방식 (하이브리드 모델)
  • 정밀 프라이싱을 위해 필요한 추가 데이터:
    • 고객사 연간 Free Cash Flow (솔루션 도입 여력 파악)
    • 화공 플랜트 수주별 마진율 (현재 공개 데이터 부재)
    • 엔지니어링 단계별 인건비 (Engineering Manpower vs Construction Manpower 구분)
    • 기간 단축 시 인건비 절감액 vs 조기 완공 인센티브(보너스)
    • 리워크(재작업) 발생 시 추가 비용
    • 납기 지연 시 페널티 조항
  • 결론: 기존 공개 자료로는 정밀 프라이싱 불가 → 네트워크 기반 전문가 인터뷰 및 대표 보유 자료가 핵심 경로

💡 안건 Insight

🎯
가치 기반 프라이싱(Value-Based Pricing) 모델 적합성 높음

단순 인건비 환산이 아닌, 엔지니어링 기간 단축·리워크 감소·조기완공 인센티브를 종합한 가치 기반 가격 모델이 적합함. 이를 위해 고객사의 FCF, 프로젝트 마진 구조 파악이 선행되어야 함.

🔐
공개 데이터의 근본적 한계 → 네트워크 인텔리전스 필수

EPC 기업의 계약 구조, 페널티 조항, 실질 마진율은 비공개 영역. 대표 보유 기존 네트워크(EPC 관계자, 기업보험 전문가 등)를 통한 접근이 유일하고 효과적인 경로임.

🌐
OPEC World Oil Outlook이 수요 측면 근거자료로 활용 가능

2030~2034년 공급 부족(쇼티지) 예측 구간은 솔루션 필요성 및 시장 타이밍 근거로 활용 가능. 투자자·고객 대상 피칭 자료에 인용 권고.

⚡
BM·프라이싱은 기존 시장조사와 별개의 심층 작업 필요

현재까지의 시장조사는 탐색적 규모 파악 수준으로 BM·프라이싱 수립에는 직접 활용이 어려움. 별도 Phase로 분리하여 접근 방식과 데이터 소스를 새롭게 설계해야 함.
