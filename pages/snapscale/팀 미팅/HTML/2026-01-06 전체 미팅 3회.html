발표 구성 및 데모 검토

1.1 발표 전략: 도메인 전문성 강조

투자자 대상 피칭과 달리, 삼두종합건설은 도메인 전문가들이므로 전문 용어를 자유롭게 사용하고 구체적인 기능을 명확히 제시하는 전략을 채택했습니다.

핵심 전략

  • Auto Design for Engineers: 엔지니어를 위한 자동 디자인으로 포지셔닝
  • 실무 문제 공감: P&ID 리스트 작업, 딜리버러블 간 컨시스턴시 체크, 마크업 CAD 변환, Hydraulic, PSV 등 실제 업무 나열
  • 해결책 제시: 프롬프트만으로 모든 작업이 자동화되는 시연

1.3 협업툴 및 버전 관리

AutoFlow를 중심으로 한 통합 협업 플랫폼을 소개하며, 마크업 수정, 코멘트 히스토리, 관련 리스트 수정, 버전 관리 기능을 간단히 시연합니다.

차별화 전략

  • 오각형에서 원형으로: 기존 각 부서 간 확변 소통 → AutoFlow 중심 일방향 소통으로 커뮤니케이션 병목 최소화
  • 설계 시간 50% 이상 절감: 명확한 정량적 목표 제시
  • 스마트 P&ID 이상: P&ID, 리스트, 데이터 시트뿐만 아니라 페이스북 히스토리 시트 등 모든 딜리버러블 통합

장표 수정사항 및 피드백

2.1 오각형 다이어그램 수정

수정 내용

  • 동그라미 5개 → 6개: 기존 Equipment 대신 "장치/회전" 추가, "배관" 추가
  • 개장(改裝) 항목 추가: 설계 변경 관리 업무 반영

2.2 용어 및 표현 수정

기존 표현 수정 표현 사유
"할인" "협의 가능" 할인은 마트 같은 느낌, 전문성 저하
"환불 해줘야 돼?" 삭제 굳이 언급할 필요 없음, 후불 계약이므로
"바쁠 때/참할 때" "호황기/불황기" 더 전문적인 용어
"조달" "추가 채용" 명확성 제고
"감축" "감원" 정확한 용어 사용
"커스텀하여" "Customize" 영어 용어로 통일
"레슨런드" "Lessons Learned" 스펠링 오류 수정 (D 빠짐)

2.3 세부 포맷 수정

  • ISO Drawing: 띄어쓰기 추가, 앞 세 글자 대문자로 (ISO)
  • API 520/521: 순서대로 표기 (520, 521)
  • Orifice, In/Out: 대문자 통일 필요
  • 블루빔(Bluebeam): 스펠링 확인
  • 글자 크기: 11페이지, 10페이지 글씨가 너무 많아 가독성 저하 → 사진 위주로 재구성 고려

프레젠테이션 주의사항

  • 글자 크기 고려: 대표님 이상 고위 임원진 참석 가능성, 시력 고려하여 큰 글씨와 적은 텍스트
  • 시각 자료 우선: 정창경 발표 때처럼 사진과 이미지로 임팩트 전달
  • 3장으로 분할: 텍스트 많은 슬라이드는 각 항목별로 사진을 부각하는 방향으로 재구성

가격 전략 및 베네핏 설계

3.1 가격 책정

최종 가격 전략

  • POC 가격: 3,000만원 (4주 테스트)
  • 정식 계약 가격: 5,000만원 (1년 기준)
  • 착수비/완수금 분할: 후불 계약이지만 착수비와 완수금으로 나누어 리스크 분산
  • POC 차감: POC에서 지불한 3,000만원은 정식 계약금에서 차감

가격 정당화 논리

AspenTech 1 copy 연간 비용이 3,000~4,000만원임을 고려할 때, SnapScale의 통합 솔루션은 여러 소프트웨어를 대체하므로 5,000만원은 합리적

3.2 베네핏 설계: 할인 vs 레슨런드 커스텀

베네핏 옵션 비교
graph TB A[초기 계약 베네핏] --> B[Option 1: 금액 할인] A --> C[Option 2: 서비스 추가] B --> D[3년간 정책금액 할인] B --> E[회사 돈이라 감흥 적음] C --> F[레슨런드 커스텀 무료/50% 할인] C --> G[실질적 가치 제공] style A fill:#1e3c72,color:#fff style B fill:#ffebee style C fill:#e8f5e9 style F fill:#c8e6c9 style G fill:#c8e6c9

최종 결정: 서비스 추가 베네핏

  • 금액 할인의 한계: "회사 돈으로 소고기 먹어도 감흥 없는 것처럼" 금액 할인은 임팩트가 약함
  • 커스텀 개발 베네핏: 제너럴 솔루션 사용 + 레슨런드 커스텀 개발 50% 할인 또는 무료 제공
  • 장표 전략: 3,000만원을 노란색으로 칠해두고, 쉬는 시간에 내용 조정 가능하도록 준비
  • 협의 사항: 분위기 보고 담배 피러/커피 마시러 나가는 타이밍에 수정

핵심 논리

"기업 입장에서는 지불할 수 있는 금액대 내에 있느냐가 결정 요소. 할인은 기분 좋은 정도. 최종 금액이 지불 가능 범위 내에 있으면 서비스 추가가 더 효과적."

정확도 보장 및 기술적 신뢰성

4.1 정확도 보장 장표 추가 필요성

박규호 부장님 지적: 기존 장표가 정확도를 충분히 설명하지 못함. 중간중간 설명하면 의문이 완전히 해소되지 않은 채 넘어갈 위험.

해결 방안

  • 별도 장표 추가: 기능 설명 후 마지막에 "정확도 100% 보장" 장표로 의문 해소
  • 온톨로지 강조: 계산 코드, 롤앤레귤레이션 기준을 온톨로지로 정확하게 구별하고 반영
  • HTML 아레나 활용: 베슬 코드, PSV 코드, 배관 코드 등이 잘 정리된 자료 기반으로 설명

4.2 온톨로지 기반 정확도 확보

정확도 보장 체계
graph TD A[계산 기준/롤앤레귤레이션] --> B[온톨로지 구성] B --> C[코드화/구조화] C --> D[AI 자동화 엔진] D --> E[정확도 검증] E --> F[95% 이상 정확도] style A fill:#ffebee style B fill:#e1f5fe style C fill:#f3e5f5 style D fill:#e8f5e9 style E fill:#fff9c4 style F fill:#c8e6c9

95% 딜레마

박규호 부장님: "95%가 돼도 안 쓴다고 했잖아요. 5% 때문에 다시 확인해야 되는 순간 일이 두 번 되기 때문에."

해결책: 각 사가 가지고 있는 계산 기준을 반영하고, 온톨로지 개념도를 명확히 제시하여 신뢰 확보

4.3 데이터 보안

  • 역설계 우려: AI + 설계도 = 역설계 가능성 우려 예상
  • 데이터 보안 장표 준비: 데이터 암호화, 접근 제어, 역설계 불가능 구조 설명
  • 예상 질문 대응: Appendix에 데이터 보안 관련 상세 자료 준비

브랜딩 및 세부 사항

6.1 브랜드 컬러: 연두색 vs 파란색

연두색 선택 배경

  • 기후위기 메시지: 그린의 기운으로 기후위기 대응 강조
  • 혁신감: 그라데이션으로 혁신적 느낌 표현

우려 사항

  • 힘 빠지는 느낌: 연한 연두색이 신뢰감 부족
  • 업계 관행: 신뢰감 필요한 B2B는 보통 파란색(AspenTech, Autodesk 등)
  • 결론: 명분(기후위기)이 있으므로 현행 유지, 향후 재검토 가능

6.2 비주얼 레퍼런스 준비

조선일보 기사 등 언론 보도 스크린샷을 마지막 슬라이드에 배치하여 "한국 화학공학을 불화시키고자 하는 열망" 강조

링크드인 바이럴 활용

  • 전세계 300~600명씩 공유되는 링크드인 포스트 언급
  • "전세계 엔지니어들이 이미 다음 스텝(개념/기본 설계)으로 가려고 움직이는 상황"
  • "물고기들이 모여서 움직이는 상황에서 그들이 원하는 밥을 우리가 만들고 있다"
  • 타이밍: 적세적소에 자연스럽게 언급 (너무 강하게 밀지 말 것)

8.1 핵심 인사이트

1. 고가 전략의 중요성

AspenTech 등 업계 표준 소프트웨어가 3,000~4,000만원인 시장에서 5,000만원은 오히려 신뢰감을 준다. 너무 저렴하면 품질에 대한 의구심 발생.

2. 베네핏의 실질적 가치

"회사 돈으로 소고기 먹어도 감흥 없는 것처럼" 금액 할인보다 레슨런드 커스텀 무료 제공 등 실질적 서비스가 더 효과적이다.

3. 95%의 함정

정확도 95%는 의미 없다. 5% 때문에 재확인해야 하면 일이 두 배가 되므로 오히려 역효과. 거의 100%에 가까운 정확도를 보장하거나, 명확한 검증 프로세스를 제시해야 한다.

4. 첫 고객의 전략적 가치

삼두와의 계약은 단순 매출이 아니라 검증된 레퍼런스 확보다. 향후 포스코, 현대건설, GS건설 등 대기업 영업에서 결정적인 신뢰 자산이 될 것이다.

5. 글로벌 트렌드 활용

링크드인에서 전세계 엔지니어들이 AI 자동화를 논의하고 있는 상황을 언급하여 "우리만의 이야기가 아니라 글로벌 흐름"임을 강조. 이는 삼두가 먼저 움직이지 않으면 경쟁에서 뒤처질 수 있다는 FOMO 자극.
