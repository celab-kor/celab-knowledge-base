1. 삼두건설 POC 제안서 전략

▼

논의 내용:

기존 PPT 자료에서 가격 부분만 제외하고 발송하는 방안에 대해 논의했습니다. 김상윤 대표는 데모 중심의 자료가 효과적일 것으로 판단했으나, 박규호 컨설턴트는 텍스트 문서 형태로 먼저 접근하는 것이 더 적합하다고 제안했습니다.

주요 논점:

  • POC 단계에서는 완성된 제품이 아닌 기술 구현 가능성을 보여주는 것이 목적
  • 워크플로우 중심의 텍스트 설명 + 동영상 데모 조합이 효과적
  • 가격은 별도로 제시하되, 초기 제안서에서는 제외
  • 데모 영상의 링크 오류 발견 및 정정 메일 발송 필요

전략적 인사이트

  • 문서 형식 선택의 중요성: 스타트업의 POC 제안은 완성도보다 기술 구현 가능성에 초점을 맞춰야 하며, 텍스트 중심 문서가 기업 구매 의사결정자에게 더 친숙한 형태입니다.
  • 가격 정보의 타이밍: 초기 제안서에서 가격을 제외함으로써 기술적 가치에 먼저 집중하게 하고, 관심 확보 후 가격 협상 여지를 남겨둡니다.
  • 데모의 중요성: AutoCAD 자동 마크업 기능 등 실제 작동하는 기술을 시연함으로써 "CES의 LG 빨래 로봇"처럼 임팩트를 줄 수 있습니다.

2. 핵심 기술 가치 제안 및 차별화 전략

▼

기술적 핵심 가치:

SnapScale의 경쟁 우위는 온톨로지(Ontology) 기반 자체 구축 AI 모델에 있습니다. 단순히 ChatGPT나 범용 LLM을 활용하는 것이 아니라, 화학공학 도메인 특화 지식 베이스를 구축했다는 점이 가장 중요한 차별화 포인트입니다.

P&ID 인식 기술:

  • 현재 정확도: 90% 이상 (내부 데이터 기준 99% 달성)
  • POC 목표: 현장 데이터 기반 95-99% 정확도 구현
  • 기술 차별화: 기존 OCR 방식 대비 우수한 인식률 (SnapScale 독자 기술)

전체 워크플로우 자동화:

  • Requirement → PFD/HMB → Calculation → Excel 자동 생성 → Word 데이터시트
  • P&ID와 Aspen HYSYS API 연동
  • AutoCAD 자동 마크업 기능 (실시간 주석 자동 생성)
  • Hydraulic Calculation 등 복잡 계산 자동화

AutoCAD 자동화 데모:

회의 중 시연된 AutoCAD 마크업 자동화 기능은 P&ID 도면에 누락된 주석을 AI가 자동으로 인식하고 생성하는 기능입니다. 속도는 최적화가 필요하지만, 기술 구현 가능성을 명확히 보여주는 임팩트 있는 데모로 평가되었습니다.

전략적 인사이트

  • 도메인 전문성이 핵심: 범용 AI 기술이 아닌 화학공학 특화 온톨로지가 진입장벽이자 경쟁우위입니다. 기술 구현 방법은 공개하지 않되, 온톨로지 구축 능력은 강조해야 합니다.
  • 인식률 90%의 의미: 현재 시장의 OCR 기반 솔루션 대비 우수하며, POC를 통해 95-99%까지 향상 가능하다는 로드맵이 명확합니다.
  • 통합 솔루션의 가치: 개별 작업 자동화가 아닌 전체 워크플로우 통합이 진정한 가치입니다. 한국조선해양의 사례처럼 P&ID 속성값을 입력할 여력이 없던 기업들에게 획기적인 솔루션이 될 수 있습니다.

4. 가격 전략 및 비즈니스 모델

▼

POC 및 본계약 가격 구조:

💰 기본 가격 정책

  • POC 비용: 3천만원
  • 본계약: 5천만원/년/프로젝트
  • 비교 기준: AspenTech HYSYS 스탠드얼론 라이선스 약 5천만원/년 수준

🎯 토큰 기반 라이선스 모델

AspenTech 모델 벤치마킹:

  • AspenTech는 토큰제 운영: 예) 27개, 60개 단위로 토큰 구성
  • 각 기능별 토큰 소비량 차등화 (HYSYS 기본: 18토큰, Dynamic 추가: 6토큰 등)
  • 워크북(Excel 연동) 등 부가 기능도 별도 토큰

SnapScale 토큰 모델 설계 방향:

  • 프로젝트 기반: 1년에 5천만원으로 약 1-2개 프로젝트 수행 가능
  • 팀 단위 구성: 한 프로젝트당 6개 discipline (기계, 공정, 배관, 계장, 전기, 구조/토목) 팀 구성
  • IP 기반 라이선스: 개인 PC IP 단위로 접근 허용 (AspenTech는 1대 PC 스탠드얼론)
  • 토큰 소비 체계:
    • 프로젝트 개시 시 토큰 차감 (예: 20토큰 계약 시 6토큰 소비)
    • 프로젝트 종료 시 토큰 반환, 다른 팀 사용 가능
    • 백엔드 데이터는 항상 구동되어야 하므로 추가 비용 발생
    • 토큰 수량 증가 시 할인 혜택

기능별 토큰 차등화 (예시):

  • 베이직: 온톨로지 기반 화공 LLM만 사용 (최소 토큰)
  • 스탠다드: Requirement → Aspen HYSYS API 연동 → PFD/HMB 생성
  • 프리미엄: Hydraulic Calculation, PSV 자동 계산 등 고급 기능 포함

📚 교육 프로그램 제공 전략

AspenTech의 SC(Solution Consultant) 모델을 벤치마킹한 교육 서비스 제공:

  • 온사이트 워크샵: 본계약 체결 시 SC 주기적 방문, 2회까지 반나절-하루 워크샵 무상 제공
  • 정규 교육 토큰: 대형 계약 시 무상 교육 시트(Seat) 제공
    • 예) 100시트 = 3일 강의 2회 수강 가능
    • 약 10명 × 3일 교육 규모
  • 교육 내용: 전체 워크플로우에 대한 이론 교육 (화공 기초부터 실무 적용까지)
  • 차별화 포인트:
    • 단순 소프트웨어 사용법이 아닌 화공 도메인 교육
    • 신입/주니어 엔지니어의 빠른 실무 적응 지원
    • 노하우 전달 및 지식 서스테이너빌리티 확보
    • Competency Level 향상 및 Consistency 유지

⚠️ 가격 경쟁력 고려사항

삼두건설 피드백:

  • AutoCAD → ZW-CAD 전환 (비용 절감 목적)
  • MS Excel AI 기능을 위한 Enterprise 구독제 비용 부담 (구독제 전환 필요, 최신 버전 요구)
  • 삼두 측에서 "기존 가격의 절반" 농담처럼 언급 (구체적 제안은 아님)

대응 전략:

  • 가격 유지: 현재 가격은 절대 비싸지 않음 (박규호 의견)
  • 가치 명확화: MS Excel AI는 단순 함수 도움 수준, SnapScale은 전체 워크플로우 자동화
  • 기존 소프트웨어 대체 아닌 보완:
    • Aspen HYSYS는 이미 Excel 연동 기능 보유
    • SnapScale은 Hydraulic Calculation 등 백그라운드 계산을 자동화하여 Excel에 결과만 출력
    • MS Copilot 대비 압도적인 도메인 특화 기능
  • ROI 제시: 인건비 절감, 설계 시간 단축, 정확도 향상을 통한 총소유비용(TCO) 절감 효과 강조

전략적 인사이트

  • 토큰 모델의 전략적 가치: AspenTech처럼 토큰 기반 모델은 고객사의 확장성을 보장하면서도 SnapScale의 수익 다각화를 가능하게 합니다. 초기 POC 성공 후 자연스럽게 토큰 추가 구매로 이어지는 구조입니다.
  • 교육 프로그램의 차별화: 소프트웨어 판매를 넘어 "엔지니어 역량 강화 파트너"로 포지셔닝함으로써 장기적 고객 관계를 구축할 수 있습니다. 이는 단순 비용 절감 툴이 아닌 "비즈니스 성장 도구"로 인식되게 합니다.
  • 가격 정당성 확보: MS Excel AI나 범용 도구와의 직접 비교를 피하고, 도메인 특화 가치와 전체 ROI 관점에서 접근해야 합니다. 현재 가격(POC 3천만원, 연간 5천만원)은 AspenTech 수준으로 시장에서 합리적입니다.
  • 프리미엄 포지셔닝 유지: 삼두의 가격 인하 압박에 대해서는 "농담" 수준으로 받아들이고, 오히려 기술적 가치와 정확도 보장을 강조하여 프리미엄 이미지를 유지해야 합니다.

장기 비전 제시:

  • 국내 POC → 국내 대기업 본계약 → 글로벌 B2B 진출
  • 커스터마이징 수주: 고객사별 팀 구성 (개발자+엔지니어 풀)
  • 플랫폼 확장: 화공 플랜트 → 다른 산업군 (조선, 석유화학 등)

전략적 인사이트

  • POC-First 접근의 중요성: 투자를 먼저 받는 것보다 2-3개의 검증된 POC를 완료한 후 투자를 받는 것이 기업가치 협상에서 월등히 유리합니다. 현재 시점은 투자를 서두르기보다 POC 성과에 집중해야 합니다.
  • 현대엔지니어링 효과: 현대엔지니어링이 먼저 연락한 것은 투자자들에게 매우 긍정적인 신호입니다. "수요처가 먼저 찾아오는 기술력"을 입증하는 사례이기 때문입니다.
  • 3-4개월 시간 절약의 가치: 박규호 컨설턴트의 평가처럼 현재 영업 진행 속도는 계획보다 3-4개월 앞당겨진 상태입니다. 이는 투자 유치 타임라인도 앞당길 수 있음을 의미합니다.
  • CVC 전략적 활용: 두산이나 GS의 CVC와 POC를 진행하면, 단순 고객을 넘어 전략적 투자자로 전환될 가능성이 있습니다. 이는 시장 검증과 자금 확보를 동시에 달성하는 효율적 방법입니다.
  • 조직 확장 시점: 투자 유치 후 엔지니어를 보강함으로써 기술적 디테일과 정확도를 높여 POC를 본계약으로 전환하는 성공률을 높일 수 있습니다.
