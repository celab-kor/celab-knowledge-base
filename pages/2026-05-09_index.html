    ⏱ 0:50 – 1:00 (10분)

    서버(Server)란 무엇인가?

    카페를 예로 들면: 손님(Slack)이 주문(메시지)을 하면, 바리스타(Worker)가 받아서 커피머신(Google Calendar)에 명령을 내립니다. 서버 = 바리스타. 항상 켜있고, 요청을 기다립니다.

    graph LR A["💬 Slack\n(손님)"] -->|"메시지 전달"| B["☁️ Worker\n(바리스타)"] B -->|"일정 추가 요청"| C["📅 Google Calendar\n(커피머신)"] C -->|"완료"| B B -->|"결과 알림"| A style B fill:#f97316,color:#fff

    왜 Cloudflare Worker인가?

    💰

    완전 무료

    하루 10만 건 요청까지 무료. 우리 용도로는 충분하고도 남습니다.

    sequenceDiagram participant U as 👤 사용자 participant SL as 💬 Slack participant SA as 🔌 Slack App participant W as ☁️ Worker U->>SL: 채널에 메시지 입력 SL->>SA: 이벤트 발생 (message 이벤트) SA->>W: HTTP POST (JSON) Note over SA,W: X-Slack-Signature 헤더로 인증 W->>SA: 200 OK W->>SL: chat.postMessage API 호출 SL->>U: 봇 응답 표시

    로컬 PC에서 실행하는 Python 스크립트에서 Slack Bot Token 같은 비밀 값을 관리하는 방법은 OS별로 다릅니다. (Worker에 올라간 코드는 wrangler secret으로 따로 처리합니다.)

    🍎 macOS — Keychain

    macOS의 키체인(Keychain)에 저장하면 암호화되어 안전합니다. 터미널에서 한 번만 저장하면 됩니다.

    터미널 — Keychain에 저장
    # Slack Bot Token 저장
security add-generic-password \
  -s "slack-bot-token" \
  -a "mybot" \
  -w "xoxb-YOUR-TOKEN-HERE"

# 저장됐는지 확인
security find-generic-password -s "slack-bot-token" -w
    Python — Keychain에서 읽기
    import subprocess

def get_secret(service):
    result = subprocess.run(
        ["security", "find-generic-password",
         "-s", service, "-w"],
        capture_output=True, text=True
    )
    return result.stdout.strip()

slack_token = get_secret("slack-bot-token")

    Slack 서명 검증 코드

    가짜 요청을 막기 위해 Slack이 보낸 메시지가 맞는지 확인합니다.

    src/index.js — 서명 검증
    async function verifySlackSignature(request, body, signingSecret) {
  const timestamp = request.headers.get('x-slack-request-timestamp');
  const signature = request.headers.get('x-slack-signature');

  // 5분 이상 지난 요청은 거부 (리플레이 공격 방지)
  if (Math.abs(Date.now() / 1000 - timestamp) > 300) return false;

  const baseString = `v0:${timestamp}:${body}`;
  const key = await crypto.subtle.importKey(
    'raw', new TextEncoder().encode(signingSecret),
    { name: 'HMAC', hash: 'SHA-256' }, false, ['sign']
  );
  const sig = await crypto.subtle.sign('HMAC', key, new TextEncoder().encode(baseString));
  const expected = 'v0=' + [...new Uint8Array(sig)].map(b => b.toString(16).padStart(2, '0')).join('');
  return expected === signature;
}

    Claude Tool Use 호출 예시
    async function dispatch(text, env) {
  // 1. 사용 가능한 기능(Tools) 정의
  const tools = [
    {
      name: 'sync_company',
      description: '회사 SharePoint 일정을 CELab 캘린더에 동기화',
      input_schema: { type: 'object', properties: {} }
    },
    {
      name: 'create_event',
      description: '새 일정을 직접 생성. "5월 26일 오전 팀 미팅 추가"',
      input_schema: {
        type: 'object',
        properties: {
          title: { type: 'string', description: '일정 제목' },
          date: { type: 'string', description: '날짜 (YYYY-MM-DD)' },
          ampm: { type: 'string', enum: ['am', 'pm', 'all'] }
        }
      }
    },
    // ... 더 많은 tools ...
  ];

  // 2. Claude에게 메시지와 tools 전달
  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'x-api-key': env.ANTHROPIC_API_KEY,
      'anthropic-version': '2023-06-01',
      'content-type': 'application/json'
    },
    body: JSON.stringify({
      model: 'claude-haiku-4-5-20251001', // 빠르고 저렴한 모델

      tools,
      messages: [{ role: 'user', content: text }]

  });

  // 3. Claude가 선택한 Tool 실행
  const data = await response.json();
  const toolUse = data.content.find(c => c.type === 'tool_use');

  switch (toolUse.name) {
    case 'sync_company':  return handleCompanySync(env);
    case 'create_event':  return handleCreateEvent(toolUse.input, env);
    // ...
  }
}
