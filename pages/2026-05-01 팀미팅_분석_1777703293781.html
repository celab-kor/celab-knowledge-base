🌊 Hydraulic 계산 — 구간 분할 및 Equivalent Length 이종현 부장 답변
Q1: 구간 분할 기준

고도(Elevation) 기준 가장 높은 지점에서 구간 분할. 밸브·Flowmeter는 별도 구간 처리.

Control Valve·Flowmeter의 ΔP는 Equivalent Length에 미포함 → 전 구간 압력에서 직접 차감 방식 적용
Q2: 엘보우 구간 귀속

구간 경계에 걸친 엘보우는 어느 구간에 넣어도 무관. 단, Isometric Drawing 표현 기준으로 처리하면 추후 혼동 방지.

Q3: Elevation 오차

파이프 치수 합산값(1,922mm)과 고도 차이값(1,780mm) 불일치(약 140mm) → 휴먼 에러. 약 14~15cm 오차로 무시 가능 수준.

Q4: Isometric 점선 의미

점선 = 해당 페이지에서 미고려 항목 (다음 페이지에서 처리). 실선만 해당 구간 산출에 포함.

⚡ PSV API 문서 해석 — 조동사 의무도 이종현 부장 답변
용어의무 수준설명
SHALL 최고 강제 협의 여지 0%. 지구가 망해도 반드시 해야 함. MUST보다 상위.
MUST / HAVE TO 강제 반드시 해야 함. SHALL보다 약간 낮은 강제성.
SHOULD 권고 해야 하는 것이나 일부 예외 가능성 있음.
MIGHT 선택적 협의 가능. 하는 것이 좋으나 안 할 수도 있음.

🏭 필즈 인터뷰 신규 용어 — HVAC·ELEC Load List 사전 학습 정보
HVAC

Heating, Ventilation & Air Conditioning의 약자. 실내 온도·습도를 일정하게 유지하는 시스템. 반도체 공장·식품 공장 등 환경 제어 필수 시설에 적용.

Electrical Load List / SLD

전기팀 산출물. 전기 부하 목록(Load List) + Single Line Diagram(SLD). 견적 시 전기 물량 산출 근거 문서.

LD (Liquidated Damages)

제기된 우려

한울(기계설계)·필즈(해양플랜트 견적)·화학안전원·PKU 등 업종이 상이. 선택과 집중이 필요한 것 아닌가?

대표 답변 (상윤)

"공정 설계만으로는 지속가능 사업 불가. 전주기 타겟팅 필수. 들어온 건이 모두 적극적·협력적 파트너 → 인력 추가로 흡수하며 확장이 최선. '먹을 거 미리 메맞자' 전략."

이종현 부장 관점 정리
  • 한울: 리소스 최소 → 온보딩 후 화공플랜트 공정 설계로 연계
  • 화학안전원: 화공플랜트 레귤레이션 온톨로지 → 화학안전 온톨로지 확장 (기술적 연속성)
  • 필즈: 견적 시작 → 해양플랜트 전체로 확장 + 투자 전 매출 레퍼런스 확보
  • 제품화·사업화: 투자 후 개발자 채용 + 이을범 교수 IP 제휴로 가속화 필요
